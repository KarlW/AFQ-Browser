afqbrowser.mat2tables¶

afqbrowser.mat2tables(mat_file_name, subject_ids=None, stats=None, out_path=None)¶

Create a nodes table and a subjects table from an AFQ .mat file

Parameters:

mat_file_name : str

Full path to an AFQ-processed mat-file

subject_ids : list, optional

Identifiers for the subjects. Default: [‘subject_001’, ‘subject_002,’ …]

stats : list, optional

List of keys for statistics to pull from the AFQ data. Default: pull out all of the statistics that are in the mat file.

out_file : str

Full path to the CSV file to be saved as output
Returns:

tuple: paths to the files that get generated: (nodes, subjects) :
